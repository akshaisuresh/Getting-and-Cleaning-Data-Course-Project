## How to Run

1. Clone or download this repository.
2. Open R (or RStudio) and set your working directory to the repo root.
3. Source the script:

```r
source("run_analysis.R")
```

If the `UCI HAR Dataset` folder is **not** present, the script automatically downloads and unzips the data. Otherwise it uses the existing folder.

The script requires the `dplyr` package (`install.packages("dplyr")` if needed).

Output is written to `tidy_data.txt` in the working directory.

## Script Walkthrough (`run_analysis.R`)

### Step 0 — Auto-download
Checks for `UCI HAR Dataset/`; downloads and unzips from the Coursera CDN if absent.

### Step 1 — Merge train and test sets
Reads `X_train.txt`, `y_train.txt`, `subject_train.txt` and their `test` equivalents, column-binds each partition, then row-binds both into one data frame of **10,299 rows × 563 columns**.

### Step 2 — Extract mean and standard deviation columns
Keeps only columns whose names contain `mean()` or `std()` (66 measurement columns), plus `subject` and `activity`, giving **68 columns**.

### Step 3 — Descriptive activity names
Replaces integer activity codes (1–6) with the labels from `activity_labels.txt` (e.g., `WALKING`, `LAYING`).

### Step 4 — Descriptive variable names
Applies a chain of `gsub()` calls to expand abbreviations:
- `t` → `time`, `f` → `frequency`
- `Acc` → `Accelerometer`, `Gyro` → `Gyroscope`, `Mag` → `Magnitude`
- Removes dots and parentheses; capitalises `Mean` / `Std`

### Step 5 — Independent tidy summary
Groups by `subject` and `activity`, computes the **mean** of each of the 66 measurement columns, and writes the resulting **180 × 68** data frame to `tidy_data.txt` using `write.table(..., row.names = FALSE)`.

## Tidy Data Principles Met

- **One variable per column** — each column holds exactly one measurement type.
- **One observation per row** — each row is one subject-activity combination.
- **One table per observational unit** — all summary data is in a single file.
- **Descriptive names** — no abbreviations remain in column or activity names.
